# Не-Матрица
 
> Предпочитаю использовать хеш-мапы, так что никаких матриц!

Единственное текстовое рпг, где можно наладить нарко-бизнес, или стать президентом. Ну или можно будет... наверно... если будет настроение делать.
Использую собственный экспериментальный модуль vivid_mud - тонкую обертку над RPyC и **textual serve** - кстати тоже экспериментальным.
Стабильность и производительно... кхм, нет времени, тем более для столь обширной аудитории.
